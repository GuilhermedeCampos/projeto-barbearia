# projeto-barbearia
Projeto de desenvolvimento de um site profissional para uma barbearia.

## Tecnologias usadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Página inicial
- Serviços
- Galeria de cortes
- Área de contato
- Design responsivo

  ## Objetivo
Criar uma presença online para apresentar a barbearia e os serviços prestados pelo barbeiro
